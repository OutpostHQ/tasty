# tastycss

**tastycss** package is a framework-agnostic implementation of [TastyCSS](https://tasty.numl.design), a CSS-in-JS library.

<p><br /></p>

```sh
# with npm
npm install tastycss

# with yarn
yarn add tastycss
```
